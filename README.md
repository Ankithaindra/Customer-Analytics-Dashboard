# 📊 Customer Analytics Dashboard

## 📌 Project Overview
This project focuses on analyzing customer behavior using Power BI.  
It provides insights into customer segmentation, spending patterns, and product performance.

---

## 🎯 Objectives
- Analyze customer purchasing behavior  
- Identify high-value customers  
- Understand product category performance  
- Visualize key business metrics  

---

## 🛠️ Tools & Technologies
- Python (Data Cleaning & Analysis)
- SQL (Data Extraction)
- Power BI (Dashboard & Visualization)

---

## 📂 Dataset Features
The dataset includes:
- Customer_ID  
- Age  
- Gender  
- Product_Category  
- Purchase_Amount  

---

## 📊 Key KPIs
- Total Customers  
- Total Revenue  
- Total Purchases  
- Average Purchase Value  

---

## 📈 Dashboard Features
- Customer Segmentation (New, Regular, Loyal)
- Sales by Product Category  
- Gender-based Spending Analysis  
- Purchase Amount Distribution  
- Interactive Filters (Slicers)

---

## 📷 Dashboard Preview
![Dashboard](dashboard.png)

---

## 🔍 Key Insights
- Majority of customers fall into the "Regular" segment  
- Certain product categories contribute more to revenue  
- Spending patterns vary across gender groups  
- Most purchases are within a specific price range
